## 🚀 Rapid Deployment

We have unified the startup process. No need to manage multiple terminals manually.

### 1️⃣ Clone and Initialize
```bash
git clone https://github.com/your-repo/Smart-CMS.git
cd Smart-CMS
```

### 2️⃣ Global Installation
Install all dependencies for root, client, and server in one command:
```bash
npm run install:all
```

### 3️⃣ Environment Configuration
Create a `.env` file in the `/server` directory:
```env
MONGO_URL=mongodb://127.0.0.1:27017/smartcms
PORT=5000
JWT_SECRET=your_secure_vault_key
```

### 4️⃣ Launch Command Center
```bash
npm start
```
*Wait for the client (Port 3000) and server (Port 5000) to synchronize.*
